## 프로젝트 소개
주특기언어로 강의를 듣고 CRUD구현
주제는 자유여서 항해에 관련된 주제로 정하면 좋을 것 같아서 결정.

항해에 대해 불만이라던가 좋은점을 익명으로 작성해서 부담없이 소통할 수 있다.
